## Deploy on GitHub Pages

For your current repository:

1. Open your `Haris-Portfolio` GitHub repo.
2. Delete old broken build files if needed.
3. Upload all files from this folder into the repo root.
4. Go to **Settings → Pages**.
5. Set source to **Deploy from branch**.
6. Select branch: `main`.
7. Select folder: `/root`.
8. Save.

Your site should open at:

```text
https://haris-byte.github.io/Haris-Portfolio/
```

## Local preview

Open `index.html` directly in your browser.

For a cleaner local server:

```bash
python -m http.server 5173
```

Then open:

```text
http://localhost:5173
```
